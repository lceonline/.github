**LCE Online** is a community-driven networking project focused on preserving, enhancing, and expanding **Minecraft Legacy Console Edition**. The goal is to restore and extend how LCE's multiplayer worked — bringing back Authentication, Proxied Worlds/Servers, and Global Leaderboards.


This project was started on **3/4/26**, making it one of the oldest LCE projects. It is also known as **LCEOnline**, **LEMP**, and **MCLEMP**.

---

## Goals

We aim to bring these features to Legacy Console Edition:

- Networking features from PlayStation Network / XBOX Live
- Secure user-to-server authentication and real player profiles
- Bring communities and players together
